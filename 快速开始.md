# 🚀 快速开始指南

## 📥 第1步：下载APK

访问：https://github.com/MSG-change/GrabOrderApp/actions

1. 点击最新的 **"Build Android APK"** 工作流（绿色✅）
2. 滚动到底部 **Artifacts** 区域
3. 下载 `graborder-apk.zip`
4. 解压得到 `GrabOrder-*-debug.apk`

---

## 📱 第2步：安装到手机

### 方式A：通过USB

```bash
adb install -r GrabOrder-*-debug.apk
```

### 方式B：直接传输

1. 将APK传到手机
2. 点击安装
3. 允许"未知来源"安装

---

## 🎮 第3步：使用APP

### 方式1：手动Token（最简单）

```
1. 使用Charles/HTTP Canary抓包获取Token
2. 打开抢单助手APP
3. 粘贴Token到输入框
4. 点击"保存Token"
5. 点击"启动抢单"
```

### 方式2：VPN抓包（自动）

```
1. 打开抢单助手APP
2. 开启"VPN自动抓包"开关
3. 授予VPN权限（弹窗点"确定"）
4. 打开目标App并登录
5. 返回抢单助手，Token自动填入
6. 点击"启动抢单"
```
