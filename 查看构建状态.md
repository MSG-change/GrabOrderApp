# 查看APK构建状态

## 快速查看

1. **打开GitHub Actions**
   - 访问: https://github.com/MSG-change/GrabOrderApp/actions
   - 点击最新的构建任务

2. **查看构建进度**
   - 绿色 ✓ = 成功
   - 黄色 ◐ = 进行中
   - 红色 ✗ = 失败

3. **下载APK**
   - 构建完成后，在页面底部找到 "Artifacts"
   - 下载 `graborder-apk` 压缩包
   - 解压后得到 `.apk` 文件

## 构建时间

- **首次构建**: 30-60分钟（下载依赖）
- **后续构建**: 20-40分钟

## 如果构建失败

查看构建日志，常见问题：
- 网络超时（会自动重试）
- 依赖下载失败
- 配置错误

## 构建成功后

1. **下载APK** 到电脑
2. **传输到手机**（USB/微信/QQ等）
3. **安装APK**
4. **查看日志**:
   ```bash
   adb logcat | grep GrabOrder
   ```
