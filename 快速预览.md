# 快速预览指南

## 三步预览

### 1. 检查依赖

```bash
cd GrabOrderApp
python3 test_preview.py
```

### 2. 安装依赖（如果缺少）

```bash
pip3 install kivy pillow requests
```

### 3. 启动预览

```bash
# 方法1：使用脚本
./preview_pc.sh

# 方法2：直接运行
python3 main.py
```

## 预览效果

预览窗口会显示：
- ✅ 完整的UI界面
- ✅ 所有按钮和输入框
- ✅ 日志显示区域
- ✅ 实时日志输出

## 注意事项

1. **VPN功能**: PC上不可用（需要Android系统）
2. **抢单服务**: 需要真实Token才能测试
3. **字体显示**: PC上可能显示方块（APK中已包含中文字体）

## 预览后

预览确认UI正常后：
1. ✅ 提交代码: `git add -A && git commit -m "预览测试通过" && git push`
2. ✅ 触发构建: GitHub Actions会自动构建APK
3. ✅ 下载APK: 从GitHub Actions Artifacts下载
4. ✅ 安装测试: 安装到手机进行真实测试

## 常见问题

### Q: 窗口太大/太小？
A: 可以拖动窗口边缘调整大小，或者按F11全屏

### Q: 中文显示为方块？
A: 这是正常的，PC预览不包含中文字体。APK中会包含字体文件。

### Q: 点击按钮没反应？
A: 检查终端日志输出，查看是否有错误信息

### Q: 如何退出预览？
A: 关闭窗口或按 `Ctrl+C`
